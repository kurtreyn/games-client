# GamesClient

- Node Version: `v22.22.3`
- Angular version `22.0.0`
- Angular CLI version `22.0.1`
- Install dependencies: `npm install --legacy-peer-deps`
- Start development server: `npm start`

Local development requires the backend server to be running. You can start the backend server by following the instructions in the [GamesSocketServer](https://github.com/kurtreyn/games-socket-server) repository.
